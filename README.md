# Frontend

Frontend starter pack for quick drafts and experiments.

## Features
* ```npm``` as build tool
* ```webpack``` to bundle assets
* ```nunjucks``` as a templating engine
* ```browsersync``` for synchronized testing

## Installation
```
npm install
```
